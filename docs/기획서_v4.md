## 1. 문서 목적

v4는 **v0.2.2 안정 릴리스 이후 완료된 팝업·카테고리·붙여넣기 UX**를 반영한 현행 기획 문서이다.

v3 대비 핵심 추가:

- **카테고리 플라이아웃**: 목록 우측에 전체 내용 패널(Top 5와 동일 축)
- **카테고리 목록 유지**: 마우스 이탈·외부 앱 이동 시에도 목록 창 유지(×로만 닫기)
- **팝업 유지 모드**: 다중 창 연속 붙여넣기 시 올바른 대상 HWND 추적
- **Win+V 제외**: 붙여넣기용 클립보드 쓰기가 Windows 클립보드 히스토리에 쌓이지 않음

v3에서 이미 완료된 ZIP·백업·단일 인스턴스·환경설정·배포 파이프라인은 v4에서도 **구현 완료**로 유지한다.

## 2. 제품 한 줄 정의

**Windows에서 텍스트·이미지 상용구를 카테고리별로 저장하고, 전역 단축키로 커서 옆 팝업을 열어 클립보드 + Ctrl+V로 붙여넣는 트레이 상주 유틸리티.**

- Shell Extension **아님**
- 흐름: **전역 호출 → 자체 팝업 → (클립보드, Win+V 미기록) → SendInput Ctrl+V → (옵션) 클립보드 복원**

## 3. 배경 및 문제

### 3.1 Pain Point

- 반복 문구·이미지를 매번 검색·복사
- OS **Win+V** 히스토리만으로는 이름·카테고리·핀 관리가 어려움
- 팝업을 연 상태에서 **여러 창**에 연속 붙여넣기할 때 대상 창이 바뀌면 실패

### 3.2 제안 가치

| 가치 | 설명 |
|------|------|
| 동선 단축 | 호출 → 검색/선택 → 붙여넣기 |
| 멀티 포맷 | 텍스트·이미지 동일 UX |
| 로컬 완결 | 서버·동기화 없음 |
| 업무 친화 | Top 5, 카테고리, 검색, 핀 |
| 클립보드 정리 | 상용구 붙여넣기가 Win+V 목록을 오염시키지 않음 |

## 5. 핵심 시나리오 (v4)

### A — Top 5

1. `Ctrl+Shift+Q`(기본) → 메인 팝업
2. Top 5 **오버** → 메인 팝업 **우측** 「Top 5 상용구 내용」
3. **클릭** → 붙여넣기  
   - `close_popup_after_paste` **OFF(기본)**: 팝업·상세 유지, 외부 창으로 이동해도 팝업 닫히지 않음  
   - **ON**: 성공 후 전체 닫힘

### B — 카테고리 (v4 UX)

1. 카테고리 **클릭** → 메인 우측 「{카테고리명} 상용구」**목록 창**
2. 목록 항목 **오버** → 목록 창 **우측** 「상용구 전체 내용」
3. 목록에서 **클릭** → 붙여넣기 (`close_popup_after_paste` OFF 시 목록 유지)
4. 목록 창은 **×**·메인 닫기·검색·다른 카테고리 선택 시에만 닫힘

### B-2 — 카테고리 목록 열린 상태에서 Top 5 (v0.2.3 최종)

1. 카테고리 목록 창이 열린 채로 Top 5에 **오버** → Top 5 상세를 **목록 창 우측(최상위)** 에 표시 (목록 창은 **닫지 않음**)
2. Top 5 **클릭** → 붙여넣기; 상세는 **Top 5 리스트 영역을 벗어날 때만** 닫힘 (붙여넣기 직후 강제 닫기 없음)
3. Top 5 사용 후 카테고리 목록에서 연속 붙여넣기 가능

### C — 팝업 유지 모드에서 다중 창 붙여넣기

1. 팝업 열림 → 초기 대상 HWND 저장
2. 사용자가 팝업/플라이아웃 위에 있을 때는 대상 유지
3. 커서가 우리 UI 밖(예: Word, 메모장)이면 **커서 아래 창** 또는 **전경 창**으로 `_target_hwnd` 갱신(100ms 폴링)
4. a → b → c 순서로 각 창에 붙여넣기 가능

### D — 관리·데이터·재실행

v3와 동일: CRUD, ZIP, 백업, 단일 인스턴스.

## 6. UI/UX

참고: `docs/기본ui-1.png`, `docs/기본ui-2.png` (목록·우측 상세 레이아웃은 v4에서 우측 2단으로 정합)

### 6.1 보조 창 정리

| 창 | 위치 | 열기 | 닫기 |
|----|------|------|------|
| Top 5 상세 | 메인 우측 (카테고리 목록 열림 시 **목록 우측·최상위**) | Top 5 오버 | Top 5 리스트·상세 영역 이탈 또는 × |
| 카테고리 목록 | 메인 우측 | 카테고리 클릭 | **×만**(마우스 이탈 ❌) |
| 카테고리 상세 | 목록 우측 | 목록 오버 | 목록·상세 영역 이탈 또는 × |

### 6.2 호버·포커스

- `close_popup_after_paste` **ON**일 때만 비활성화 후 외부 클릭 닫기 검사
- **OFF**일 때: `WindowDeactivate` 시 HWND 동기화만 수행, 팝업 유지

### 6.3 환경설정

v3와 동일: 단일 스크롤, 단축키 캡처·probe, 테마/마우스 휠 UI 없음.
