## 1. 시스템 구성

```text
main.py
  ├─ SingleInstanceGuard (QLocalServer)
  └─ AppController
       ├─ TrayController
       ├─ PopupWindow + SnippetDetailFlyout + SnippetFlyout
       ├─ ManagerWindow
       ├─ SettingsWindow + HotkeyCaptureEdit
       ├─ HelpWindow
       ├─ HotkeyService (RegisterHotKey, probe, suppress)
       ├─ PasteService / ClipboardService
       ├─ ImportExportService / BackupService / BackupScheduler
       └─ Repositories → SQLite app.db
```

## 2. FN-00 앱 공통

### FN-00-01 시작·종료

| 항목 | 내용 |
|------|------|
| 진입 | `python -m src.main`, `scripts/dev_run.ps1` |
| 초기화 | AppData, DB seed, settings.json, 로그, 트레이, 핫키 등록 |

## 4. FN-02 전역 단축키

| 항목 | 내용 |
|------|------|
| 기본값 | **`Ctrl+Shift+Q`** (`config.DEFAULT_HOTKEY`) |
| 등록 | `Win32HotkeyService.register` — HWND는 Qt 창 |
| fallback | `Ctrl+Shift+V`, `Ctrl+Alt+V`, `Ctrl+Shift+Insert`, `Ctrl+Alt+P` |
| 설정 창 | `set_suppressed(True)` — WM_HOTKEY 콜백 무시 + `open_popup` 차단 |
| probe | `probe_available`: 현재 등록과 동일 키 → OK; 아니면 unregister 후 시험 등록 |
| 파싱 | 2~3키, 수정키+일반키 (`hotkey_parser.py`) |
| UI 캡처 | `HotkeyCaptureEdit` — 클릭/Tab만 입력 모드, Esc/Del, blur finalize |

**모듈:** `hotkey_service.py`, `hotkey_capture_edit.py`, `hotkey_parser.py`

## 5. FN-03 메인 팝업

**모듈:** `popup_window.py`, `popup.qss`, `popup_flags.py`

### 5.1 레이아웃

| 영역 | 설명 |
|------|------|
| 타이틀 | DraggableTitleBar, `?` → help, `✕` → close |
| 검색 | 실시간 필터 → Top5 리스트 대체 |
| Top 5 | `snippet_repository.top_snippets(limit=5)` |
| 카테고리 | active 카테고리 목록 |
| 하단 | 추가/수정/삭제 → manager; 설정 → settings |

### 5.2 표시

| 항목 | 동작 |
|------|------|
| 위치 | `show_near_cursor(offset_px, width, height)` — settings의 width/height |
| 전경 | `capture_foreground_window()` 저장 → 붙여넣기 대상 |
| 닫기 | `close_popup_after_paste()` **true**일 때만 `WindowDeactivate` → 150ms 후 포커스 검사 닫기 |

### 5.3 Top 5 / 카테고리 / 검색

| 이벤트 | 동작 |
|--------|------|
| Top5 Enter | `SnippetDetailFlyout` (카테고리 패널 열리면 무시) |
| Top5 Click | `paste_snippet` |
| Category Click | `SnippetFlyout` |
| Category Enter | 없음 |
| Search 비움 | Top5 복원 |

### 5.4 붙여넣기 후

| `close_popup_after_paste` | 성공 시 |
|---------------------------|---------|
| false (기본) | 팝업 유지, 전경 HWND 재캡처, topmost 유지 |
| true | `_close_popup()` |

실패: 항상 경고, 팝업 유지.
