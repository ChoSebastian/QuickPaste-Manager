## 1. 문서 목적

본 문서는 QuickPaste Manager의 **제품 기획 관점**을 v2 기준으로 정리한다.  
v1 대비 **실제 구현·UI 개선·이미지 상용구·팝업 UX**가 반영되었으며, 아직 미완인 항목(배포 패키징, Import/Export 등)은 로드맵으로 구분한다.

---

## 2. 제품 한 줄 정의

**Windows에서 자주 쓰는 텍스트·이미지를 카테고리별로 저장하고, 전역 단축키로 커서 옆 팝업을 열어 즉시 붙여넣는 상주형 생산성 유틸리티.**

- Shell Extension(우클릭 메뉴) 방식 **아님**
- 흐름: **글로벌 호출 → 자체 팝업 → 클립보드 등록 → SendInput Ctrl+V**

---

## 3. 배경 및 문제

### 3.1 사용자 Pain Point

- 동일 문구·계좌정보·인사말·이메일 템플릿·로고/도장 이미지를 반복 검색·복사·붙여넣기
- 메모장, 문서, 메신저, 메일, 탐색기를 오가며 **작업 맥락이 끊김**
- 클립보드 히스토리만으로는 **이름·카테고리·고정(핀)** 관리가 어려움

### 3.2 제안 가치

| 가치 | 설명 |
|------|------|
| 동선 단축 | `호출 → 검색/선택 → 붙여넣기` 3단계 |
| 멀티 포맷 | 텍스트 + 이미지 동일 UX |
| 로컬 완결 | 서버·계정·동기화 없음 |
| 업무 친화 | Top 5, 카테고리, 검색, 핀 고정 |

---

## 4. 타깃 및 사용 환경

### 4.1 1차 타깃

- **사무직·고객응대·영업지원** 등 반복 입력이 많은 Windows 사용자

### 4.2 환경 가정

- Windows 10/11 64-bit
- Python 3.12+ 개발 / PySide6 GUI
- 일반 사용자 권한(관리자 권한 필수 아님)
- `%APPDATA%\QuickPasteManager` 로컬 저장

### 4.3 제외 대상(기획 원칙 유지)

- 팀 협업·권한·승인 워크플로
- 클라우드 동기화
- AI 생성/추천
- 암호화·보안 모듈

## 5. 핵심 사용 시나리오 (v2 반영)

### 시나리오 A — 텍스트 상용구 (Top 5)

1. Word/메신저 등에서 작업 중 `Ctrl+Shift+Q`(설정 가능) 입력
2. 커서 옆 **메인 팝업** 표시
3. **Top 5** 목록에서 항목에 마우스 오버 → 우측 **「Top 5 상용구 내용」** 패널에 전체 본문 표시
4. 내용 확인 후 메인 목록에서 항목 **클릭** → 대상 앱에 붙여넣기, 팝업 전체 닫힘

### 시나리오 B — 카테고리·이미지

1. 메인 팝업에서 **카테고리 클릭**(오버만으로는 열리지 않음)
2. 우측 **「{카테고리명} 상용구」** 패널에 축약 목록(Top 5와 동일 2줄 형식)
3. 항목 오버 → 하단 **「내용」**에 전체 텍스트/이미지
4. 이미지는 **클릭 시 원본 파일**을 기본 뷰어로 열기(팝업 유지)
5. 붙여넣기할 항목 **클릭** → 성공 시 팝업 닫힘 / 실패 시 **경고만** 표시·팝업 유지

### 시나리오 C — 관리

1. 트레이 → **상용구 관리**
2. 카테고리 CRUD, 텍스트/클립보드/파일 이미지 추가, 수정, 삭제, 휴지통 복구

## 6. UI/UX 기획 (목업 기준)

참고: `docs/기본ui-1.png`, `docs/기본ui-2.png`

### 6.1 메인 팝업

- 검색, Top 5, 카테고리, 하단 액션(추가/수정/삭제/설정 — 팝업 내 일부 비활성, 관리창 연동 예정)
- 타이틀 바: 드래그 이동, `?`, `✕`
- **항상 최상위**(HWND_TOPMOST) + 커서 인접 배치

### 6.2 보조 팝업(우측)

| 유형 | 트리거 | 제목 예 | 닫기 |
|------|--------|---------|------|
| Top 5 상세 | Top 5 오버 | Top 5 상용구 내용 | ✕ (카테고리 패널 유지) |
| 카테고리 목록 | 카테고리 클릭 | 일반 상용구 | ✕ |

### 6.3 호버·영역 규칙 (UX 정책)

- **리스트 + 연관 패널**을 하나의 호버 영역으로 간주
- 영역을 완전히 벗어나면 보조 패널 **숨김·초기화**
- 카테고리 패널이 열린 상태에서는 Top 5 오버가 **카테고리 패널을 가리지 않음**

## 11. 리스크 및 대응

| 리스크 | 대응 |
|--------|------|
| SendInput이 일부 앱에서 차단 | 경고 + 클립보드 복사 안내 |
