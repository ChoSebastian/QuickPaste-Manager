## 1. 제품 정의

QuickPaste Manager는 Windows에서 텍스트·이미지 상용구를 **카테고리별로 저장**하고, **전역 단축키**로 **커서 인접 팝업**을 호출해 **클립보드 + 자동 Ctrl+V**로 붙여넣는 **상주형 데스크톱 유틸리티**이다.

**비목표:** Shell Extension, 클라우드, 팀 기능, AI, 암호화.

## 5. 사용자 스토리 (v2)

### US-01 호출

> **As a** 사용자, **I want** `Ctrl+Shift+Q`로 팝업을 열고, **so that** 현재 작업 앱 옆에서 상용구를 고를 수 있다.

**AC**

- 단축키 후 300ms 내 팝업 표시(체감)
- 팝업은 다른 창 위에 표시
- 외부 클릭 시 팝업 닫힘(붙여넣기 중 제외)

### US-02 Top 5

> **I want** Top 5 항목에 마우스를 올리면 우측에서 전체 내용을 보고, **클릭**하면 붙여넣는다.

**AC**

- 우측 패널 제목: 「Top 5 상용구 내용」
- ✕로 우측 패널만 닫기 가능
- 성공 시 모든 팝업 닫힘

### US-03 카테고리

> **I want** 카테고리를 클릭해 상용구 목록을 보고, 항목을 고른다.

**AC**

- 카테고리 **오버만**으로는 보조 팝업 열리지 않음
- 우측 제목: 「{카테고리명} 상용구」
- Top 5 오버가 열린 카테고리 패널을 닫지 않음

### US-04 이미지

> **I want** 이미지를 미리보고, 원본을 확인하며, 지원되는 앱에 붙여넣는다.

**AC**

- 미리보기 클릭 → OS 기본 뷰어, 팝업 유지
- 메모장 등 불가 대상 → 경고, 팝업 유지
- Word/그림판 등 → 붙여넣기 성공 시 팝업 닫힘

### US-05 관리

> **I want** 관리창에서 카테고리·상용구를 등록·수정·삭제한다.

**AC**

- 텍스트·클립보드 텍스트·이미지 파일·클립보드 이미지 추가
- 휴지통 보기·복구·비우기
- 카테고리 최대 10개

## 8. 설정 항목 (`settings.json`)

| 키 | 기본값 | 설명 |
|----|--------|------|
| hotkey | Ctrl+Shift+V | 전역 단축키 |
| popup_width / popup_height | 360 / 520 | 팝업 크기 |
| popup_offset_px | 12 | 커서 오프셋 |
| paste_delay_ms | 50 | 붙여넣기 전 지연 |
| restore_clipboard_after_paste | true | 붙여넣기 후 클립보드 복원 |
| mouse_wheel_trigger_enabled | false | 미구현 |
| seed_default_categories | true | 기본 카테고리 5종 |
