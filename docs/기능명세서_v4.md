## 2. FN-03 메인 팝업 (`popup_window.py`)

### 2.1 레이아웃

v3와 동일: 검색, Top 5, 카테고리, 하단 관리/설정.

### 2.2 붙여넣기 대상 HWND

| 시점 | 동작 |
|------|------|
| `show_near_cursor` | `capture_foreground_window()` → `_target_hwnd` |
| 붙여넣기 직전 | `_sync_external_target_hwnd()` |
| 유지 모드 (`close_popup_after_paste` false) | `QTimer` 100ms → `_sync_external_target_hwnd` |

**`_sync_external_target_hwnd` 규칙:**

1. 커서가 우리 UI(메인·카테고리 목록·Top5 상세·카테고리 상세) **밖** → `capture_window_at_cursor()` (우리 HWND 제외)
2. 그 외 → `capture_foreground_window()` (우리 HWND 제외)
3. 커서가 메인 위이고 전경이 우리 팝업이면 **기존 `_target_hwnd` 유지**

**모듈:** `input_injection.capture_foreground_window`, `capture_window_at_cursor` (`WindowFromPoint` + `GA_ROOT`)

### 2.3 Top 5

| 이벤트 | 동작 |
|--------|------|
| Enter | `SnippetDetailFlyout` (카테고리 목록 열리면 무시) |
| Click | `_paste_snippet` |

### 2.4 카테고리

| 이벤트 | 동작 |
|--------|------|
| Click | `_show_category_flyout` — 메인 `rect.right()+4` |
| Enter | 없음 |

### 2.5 붙여넣기 성공 후

| `close_popup_after_paste` | 동작 |
|---------------------------|------|
| **false** (기본) | 메인 유지; **카테고리 목록·상세 유지**; topmost 재적용 |
| **true** | `_hide_category_flyout`, `_reset_top_detail`, `_close_popup` |

실패: 항상 경고, 팝업 유지.

### 2.6 보조 창 HWND·히트 테스트

- `_popup_hwnds()`: 메인 + flyout + flyout.`auxiliary_hwnds()` + top_detail
- `_auxiliary_windows_at`: flyout frame + `auxiliary_contains` (카테고리 상세)
- `_close_if_focus_lost`: 카테고리 상세 창 포커스 시 닫지 않음

## 3. FN-03a 카테고리 플라이아웃 (`snippet_flyout.py`)

### 3.1 구조

- 단일 세로 레이아웃: **DraggableTitleBar** + **QListWidget** (전체 높이)
- 하단 분할 미리보기 **제거** (v4)

### 3.2 상세 패널

- 내부 `SnippetDetailFlyout` lazy 생성
- 목록 `itemEntered` → 120ms 후 `show_snippet` at `flyout.right()+4`
- `set_header_title("상용구 전체 내용")`
- `moveEvent` / `reposition` 시 상세 위치 동기화

### 3.3 닫기 정책 (v4)

| 동작 | 카테고리 **목록** | 카테고리 **상세** |
|------|-------------------|-------------------|
| 마우스가 세 창 밖으로 이동 | **유지** | 상세만 `_reset_detail_panel` (목록·상세 영역 이탈 시) |
| 헤더 × | `hide_and_reset` + `on_close` | — |
| 메인 × / Esc / 검색 / 다른 카테고리 | popup에서 `hide` | — |
| 붙여넣기 성공 + close OFF | 유지 | 유지 |
| 붙여넣기 성공 + close ON | popup이 hide | — |

**제거됨:** `leaveEvent` → `_check_window_zone` → `hide_and_reset` (v3 이하 동작)

### 3.4 API

| 메서드 | 용도 |
|--------|------|
| `auxiliary_hwnds()` | 카테고리 상세 HWND |
| `auxiliary_contains(pos)` | 상세 hit test |
| `category_detail_widget()` | 포커스·닫기 검사 |
