# 릴리스 체크리스트

## 배포 가능 조건 (PRD §16)

- [ ] 비개발자 PC에서 설치 성공
- [ ] Windows 시작 후 트레이 상주
- [ ] 전역 단축키 호출
- [ ] 카테고리 10개까지 CRUD
- [ ] 카테고리당 ~100개 항목 탐색/검색
- [ ] 텍스트 붙여넣기 정상
- [ ] Import / Export 정상
- [ ] 로그 기록 정상
- [ ] 8시간 실행 후 치명적 종료 없음
- [ ] 제거 시 데이터 유지/삭제 선택
