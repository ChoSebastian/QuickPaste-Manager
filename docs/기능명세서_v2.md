## 2. 시스템 구성

```text
main.py → AppController
  ├─ TrayController
  ├─ PopupWindow + SnippetDetailFlyout + SnippetFlyout
  ├─ ManagerWindow
  ├─ SettingsWindow
  └─ Services: Hotkey, Clipboard, Paste, Mouse(stub), ImportExport(stub), Backup(stub)

```

## 6. FN-03 메인 팝업 (`PopupWindow`)

**모듈:** `src/ui/popup_window.py`  
**스타일:** `src/resources/styles/popup.qss`  
**창 플래그:** `Tool | FramelessWindowHint | WindowStaysOnTopHint`

### 6.1 레이아웃

| 영역 | 설명 |
|------|------|
| 타이틀 바 | QuickPaste, `?`, `✕`, 드래그 이동 |
| 검색 | 실시간 필터 |
| Top 5 | 최대 5건, 2줄 축약+썸네일 |
| 카테고리 | 카테고리명만 표시 |
| 하단 버튼 | 추가/수정/삭제/설정 — **비활성** |

### 6.2 표시·위치

| 항목 | 동작 |
|------|------|
| 트리거 | 단축키 / 트레이 |
| 위치 | 커서 + offset, 화면 밖 보정 |
| Z-order | `raise_window_topmost` |
| 전경 HWND | `capture_foreground_window()` — 붙여넣기 대상 |
| 외부 클릭 | `WindowDeactivate` → 150ms 후 포커스 검사 → 닫기 |

### 6.3 검색 (FN-03-01)

| 입력 | 동작 |
|------|------|
| 비어 있음 | Top 5 복원, 보조 패널 초기화 |
| 텍스트 있음 | `snippet_repository.search` 결과를 Top 5 리스트에 표시 |

### 6.4 Top 5 (FN-03-02)

| 이벤트 | 동작 |
|--------|------|
| `itemEntered` | 우측 `SnippetDetailFlyout` 표시 (**카테고리 패널 열려 있으면 무시**) |
| `itemClicked` | `PasteService.paste_snippet` → **성공 시만** 전체 닫기 |
| 리스트 Leave | Top5 영역(리스트∪우측상세) 벗어나면 상세 패널 reset |

**우측 상세 패널:** `SnippetDetailFlyout` (`src/ui/widgets/snippet_detail_flyout.py`)

| 항목 | 내용 |
|------|------|
| 제목 | 「Top 5 상용구 내용」 |
| 부제 | 상용구 title (📌 핀 표시) |
| 텍스트 | 전체 본문, 스크롤 |
| 이미지 | 스크롤 영역 + 클릭 시 원본 열기 |
| ✕ | 상세만 닫기 (`reset`), **카테고리 패널 유지** |

### 6.5 카테고리 (FN-03-03)

| 이벤트 | 동작 |
|--------|------|
| `itemEntered` | **없음** (오버 무시) |
| `itemClicked` | 우측 `SnippetFlyout` 표시 |

**우측 카테고리 패널:** `SnippetFlyout` (`src/ui/widgets/snippet_flyout.py`)

| 항목 | 내용 |
|------|------|
| 제목 | 「{카테고리명} 상용구」 |
| 목록 | Top5와 동일 2줄 축약(`LIST_SUMMARY_PREVIEW_LEN=50`) |
| `itemEntered` | 하단 「내용」전체 표시 |
| `itemClicked` | 붙여넣기 |
| 리스트 Leave | 하단 내용 reset |
| 창 Leave | 메인 팝업 밖이면 패널 닫기 |
| ✕ | 패널 닫기 + `selected_category_id` 해제 |

### 6.6 붙여넣기 (FN-03-04)

| 항목 | 내용 |
|------|------|
| 서비스 | `DefaultPasteService` |
| 성공 | 보조·메인 팝업 모두 닫기, usage 기록 |
| 실패 | `show_warning=True`, **팝업 유지** |
| 지연 | `QTimer.singleShot(30ms)` 후 실행 |
