## 1. 문서 목적

v3는 **실제 배포 가능 수준까지 구현된** QuickPaste Manager를 기준으로 기획·범위·UX를 정리한다.  
v2 대비 반영: **ZIP Import/Export, 자동 백업, Windows 시작 프로그램, 단일 인스턴스, 환경설정 UI 개편, 단축키 캡처·충돌 검사, 도움말 창, PyInstaller+Inno Setup**.

v2 문서에 ⬜로 남아 있던 항목 중 **구현 완료**와 **의도적 미구현·제거**를 명확히 구분한다.

---

## 2. 제품 한 줄 정의

**Windows에서 텍스트·이미지 상용구를 카테고리별로 저장하고, 전역 단축키로 커서 옆 팝업을 열어 클립보드 + Ctrl+V로 붙여넣는 트레이 상주 유틸리티.**

- Shell Extension(탐색기 우클릭) **아님**
- 흐름: **전역 호출 → 자체 팝업 → (선택) 클립보드 설정 → SendInput Ctrl+V → (옵션) 클립보드 복원**

## 3. 배경 및 문제

### 3.1 Pain Point

- 반복 문구·계좌·인사말·이메일·로고/도장 이미지를 매번 검색·복사
- 작업 앱과 메모·문서·메신저를 오가며 **맥락 단절**
- OS 클립보드 히스토리만으로는 **카테고리·핀·이름** 관리가 어려움

### 3.2 제안 가치

| 가치 | 설명 |
|------|------|
| 동선 단축 | 호출 → 검색/선택 → 붙여넣기 |
| 멀티 포맷 | 텍스트·이미지 동일 UX |
| 로컬 완결 | 서버·계정·동기화 없음 |
| 업무 친화 | Top 5, 카테고리, 검색, 핀 |

## 5. 핵심 시나리오 (v3)

### A — 텍스트 (Top 5)

1. Word/메신저 등에서 **`Ctrl+Shift+Q`**(기본, 설정 가능) 입력
2. 커서 근처 **메인 팝업** 표시
3. Top 5 항목 **오버** → 우측 「Top 5 상용구 내용」에 전문 표시
4. 항목 **클릭** → 붙여넣기  
   - 기본: **팝업 유지**(`close_popup_after_paste` 기본 OFF)  
   - 설정 ON 시: 성공 후 팝업 닫힘

### B — 카테고리·이미지

1. 카테고리 **클릭**(오버만으로는 미열림) → 우측 「{카테고리명} 상용구」
2. 항목 오버 → 하단 「내용」에 전문/이미지
3. 이미지 미리보기 **클릭** → OS 기본 뷰어로 원본 열기(**팝업 유지**)
4. 항목 클릭 → 붙여넣기(실패 시 경고, 팝업 유지)

### C — 관리·데이터

1. 트레이 또는 팝업 하단 → **상용구 관리** / **환경설정**
2. CRUD, 휴지통, 클립보드·파일 이미지 추가
3. 트레이 **보내기/불러오기** ZIP, 자동 백업

### D — 재실행

- 이미 실행 중이면 **두 번째 프로세스 종료**, 기존 인스턴스에 팝업 활성화 안내

## 6. UI/UX (목업·구현 정합)

참고: `docs/기본ui-1.png`, `docs/기본ui-2.png`, `docs/03. 환경설정창.png`

### 6.1 메인 팝업

- 검색, Top 5, 카테고리, 하단 **추가/수정/삭제/설정** → 관리창·환경설정 연동
- 타이틀 바 드래그, `?`(도움말), `✕`
- 최상위 + 커서 오프셋(`popup_offset_px`, UI 기본 노출)

### 6.2 보조 패널

| 유형 | 트리거 | 닫기 |
|------|--------|------|
| Top 5 상세 | Top 5 오버 | ✕ (카테고리 패널 유지) |
| 카테고리 목록 | 카테고리 클릭 | ✕ + 선택 해제 |

### 6.3 호버·포커스

- 리스트+연동 패널을 하나의 호버 영역으로 처리(`hover_zone.py`)
- `close_popup_after_paste`가 **켜져 있을 때만** 창 비활성화 후 외부 클릭 시 닫힘 검사

### 6.4 환경설정 (v3)

- **단일 스크롤** + Windows 스타일 토글(`SettingToggle` / `ToggleSwitch`)
- 단축키: 열 때 **저장값 표시**, **클릭 시만** 입력 모드, blur 시 미변경이면 복원
- **테마·글자 크기·마우스 휠** UI **없음**(v3에서 제거)
